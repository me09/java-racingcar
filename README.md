# java-racingcar

#### 기능목록

- [x] 각 자동차의 이름을 입력받는다.
  - [x] 출력할 때 이름을 같이 출력한다.
  - [x] 쉼표를 기준으로 구분한다.
  - [x] 5자 이하로 한다.
  - [x] 자동차 수에 제한은 없다.



- [x] 이동 횟수를 입력받는다.
  - [x] 주어진 횟수동안 자동차들은 전진 또는 정지를 한다.
  - [x] 횟수가 끝나면 우승자를 출력한다.
  - [x] 우승자는 한 명 이상일 수 있다.



- [x] 0 ~ 9 까지의 랜덤한 숫자를 통해 전진 또는 정지를 한다.
  - [x] 4 이상일 경우 전진, 3 이하일 경우 정지한다.
